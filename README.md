# Materials Science RAG Research Assistant

Domain-specific RAG system for predicting material properties and radar absorbing materials.

## Tech Stack
- **Framework:** LlamaIndex
- **Vector Store:** ChromaDB
- **Embeddings:** local sentence-transformers
- **LLM:** Ollama (Local)
- **Parsing:** Nougat / Marker
- **Evaluation:** Ragas
- **UI:** Streamlit
